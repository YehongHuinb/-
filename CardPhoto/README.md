# 关于各个文件
### 1. save 文件夹
照片保存的默认文件夹。
### 2. src 文件夹
里面有用于测试的照片、程序的介绍ppt、思维导图、界面的ui 文件和活动图等。
### 3. temp 文件夹
用于存放程序生成的中间文件(其实就一张照片)。
### 4. StartWindow.py、MainWindow.py
用PyQt uic 转换得到的界面代码。
### 5. UseViews.py
调用上述两个界面代码的的界面类。
### 6. Controller.py
处理图片的控制类。
### 7. MyButton.py
自定义按钮类，重载了鼠标按压和释放事件。
### 8. main.py
程序运行脚本。
